# shechat

A Chatting application for Women.

## Setup
* Download & Configure [Ngrok](https://ngrok.com/) for your platform
* Clone the server: `git clone https://github.com/abughalib/she-chat-server`
* `cd she-chat-server`
* Make sure your ngrok server is running and the url is configure in flutter project
* Install necessary packages `npm i`
* Start the Server `npm start`

## Build & Run
* Connect a device in Debug mode.
* Make Sure [flutter](https://flutter.dev/) is installed.
* `flutter run`
